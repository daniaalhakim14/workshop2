# MyDuit a AI-Powered Financial Advisor Application
MyDuit is an AI-Powered Personal Financial Management Application developed as part of the BITU3932 Workshop 2 course project. The goal of this project is to design a market-ready product with real-world value, addressing key financial challenges faced by many individuals.
MyDuit leverages artificial intelligence to assist users in managing their personal finances more effectively. Key features include:

- AI-Powered Budgeting
- Personalized Budget Planning
- Savings Maximization Suggestions
- Notifications on Government Welfare Programs and Financial Aid
- Financial Knowledge Content to Improve Financial Literacy

By integrating intelligent insights, educational content, and automated recommendations, MyDuit empowers users to track spending, optimize saving habits, understand financial concepts, and access support opportunities they may be eligible for—helping them work towards greater economic stability and smarter financial decisions.

# Notification Dashboard:
<img width="402" height="850" alt="image" src="https://github.com/user-attachments/assets/6f933368-4a53-4fae-8aa4-94580b1ac0c7" />
<img width="402" height="849" alt="image" src="https://github.com/user-attachments/assets/eb3d4530-e937-47fc-83c5-bb5a9d3e4566" />
Provides personalized financial notifications categorized into three color-coded types:
Green: Government programs, welfare assistance, and tax relief alerts.
Blue: Financial literacy tips, scam warnings, and educational insights.
Red: Personal finance alerts such as budget limit warnings and overspending.
•  Users can tap on each notification to view full details, eligibility information, and external references (e.g., LHDN, JKM websites).
•  Each notification is tagged with relevant icons (e.g., Healthcare, Lifestyle, Transportation) to help users quickly identify the related financial domain.
•  Notifications are automatically sorted from newest to oldest and are grouped by topic for easy browsing.
•  Supports awareness and financial literacy by empowering users with timely, reliable, and actionable information.
